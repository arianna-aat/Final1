# Portafolio Web

Este es un portafolio web diseñado para mostrar información sobre nuestro equipo, proyectos realizados, una calculadora profesional y un formulario de contacto.

## Descripción

La página web está estructurada para ofrecer una experiencia intuitiva y funcional, con los siguientes apartados principales:

- **Sobre el Equipo:** Información detallada sobre los integrantes del equipo.
- **Calculadora Profesional:** Herramienta integrada para realizar cálculos avanzados de forma sencilla y eficiente.
- **Formulario de Contacto:** Permite a los usuarios enviar mensajes o consultas directamente a través del sitio.
- **Nuestros Proyectos:** Sección donde se exhiben los proyectos desarrollados con descripciones y enlaces relevantes.
